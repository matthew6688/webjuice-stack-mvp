# Roofing Template Image Prompt Pack

Updated: 2026-05-09

Use these prompts manually in ChatGPT Image / image2 first. The goal is to test whether manual image generation can reach the reference quality before we automate it through browser control or API.

Global rules for every image:

- No text, no readable signage, no logos, no watermarks, no badges.
- Photo-realistic commercial website image, not illustration.
- Keep hands, tools, roof edges, ladders, and faces realistic.
- Avoid fake awards, review stars, licence badges, exact addresses, and brand names.
- Leave usable negative space when the image is marked as `hero`.
- Prefer warm natural light, real roof materials, and believable local-business work scenes.
- If the model adds text or a logo, regenerate.

Visual language: refined, architectural, dark blue or deep green hero mood, warm window light, editorial serif website feel, premium but still local.

## Acceptance Checklist

Keep an image only if it passes all of these:

- Looks like a real photo at website scale.
- Fits roofing specifically, not generic construction.
- Has no text/logos/watermarks/fake proof.
- Has clean composition for the intended section.
- Does not introduce claims we cannot verify.
- Could be used next to a local business CTA without feeling like obvious stock filler.
