# Productized Modern Roofing Style Guide

## Brand

Business name: Roofex Roofing Systems

Logo direction: single default demo logo using a simple geometric roof mark and readable wordmark. Do not create alternate logo choices for this template family.

## Design Tokens

```css
:root {
  --bg: #F3F4F6;
  --surface: #FFFFFF;
  --fg: #111827;
  --muted: #64748B;
  --border: #E5E7EB;
  --accent: #2F6F5E;
}
```

## Typography

- Display: Inter or system sans, large clean shapes.
- Body: Inter or system sans, neutral and readable.
- UI labels: small uppercase mono-style labels with restrained tracking.

## Layout Rules

- Lead with a contained, premium photo hero.
- Use product/material cards for roof choices.
- Use accordions for services and FAQ so the page stays compact.
- Use calm white and gray bands with charcoal typography.
- Use the green accent for primary quote actions only.
- Keep corners restrained: 8px for controls, 16px for major image panels.

## Copy Rules

- Keep language practical, clear, and customer-facing.
- Do not mention internal production terms.
- Do not invent verified business facts.
- Do not invent prices, product brands, review counts, license numbers, awards, years in business, or addresses.
- Use honest prompts such as "Use verified business contact details here" where real company data is required.
