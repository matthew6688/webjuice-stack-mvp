## Important Data Notes

- This bundle contains many generated artifacts under `clients/`, `data/`,
  `public/admin-artifacts/`, and `templates/`.
- No single file was observed above GitHub's 100 MB file limit during handoff
  preparation, but the full artifact set is large.
- The largest directory is `data/`; most of that weight is generated QA and
  template experiment evidence.
- Raw Maps scraper full dumps such as `results.full.json` are intentionally
  ignored because they can include review payloads; commit compact summaries,
  queues, reports, and hard evidence instead.
- `.env`, `.env.local`, and `.env.production` remain ignored. The settings page
  should show masked configuration only.
